Features

Vercount 是一个更快、更稳定的不蒜子计数替代方案

高速响应

公共计数链路由 Go 服务直接处理，响应更快也更稳定

准确统计

使用 POST 请求进行统计，不再使用不蒜子的过时 Referrer 方法，实现移动端 / Firefox / Safari 等浏览器的准确统计

安全可靠

保留现有兼容接口的同时，持续加强公共计数接口的安全与风控能力

自动同步

初始化自动同步所有不蒜子的数据，无需手动操作，每访问一次，数据就会自动同步

兼容性好

兼容不蒜子的 span 标签，可以无缝切换

数据可靠

使用 Redis 作为共享计数存储，数据稳定可靠，并由多个服务共同读取

清晰架构

Go + Redis 负责核心计数后端，Next.js 负责后台、鉴权、域名管理与兼容层

编辑访客数据

支持编辑访客数据，提供更灵活的数据管理选项
