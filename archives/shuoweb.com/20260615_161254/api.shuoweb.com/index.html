Phone

手机设备信息接口，返回随机机型、按品牌或系统筛选机型，以及指定型号详情。

GET /phone/

随机返回一个手机型号

GET /phone/search?brand=Apple

按品牌或系统随机返回一个结果

GET /phone/{model}

按型号查询指定设备详情

History

历史上的今天接口，返回当天或指定日期发生的历史事件。

GET /history/

返回今天的历史事件

GET /history/{date}

查询指定日期的历史事件，格式为 MM-DD

Quote

随机一句接口，返回名人名言或经典语录。

GET /quote/

随机返回一句名言

GET /quote/?author=孔子

按作者筛选随机一句
